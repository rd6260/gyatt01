## Getting Started

### Clone the Repository

```bash
git clone https://github.com/rd6260/gyatt01.git
```

### Move into the project directory:

```bash
cd gyatt01
```

### Install Dependencies
```bash
flutter pub get
```
### Run the App
```bash
flutter run
```

## Prerequisites

Make sure you have the following installed:

- Flutter SDK
- Dart
- Git
- Android Studio / VS Code
